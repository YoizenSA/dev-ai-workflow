# Skills

## Qué son

Guías especializadas que le dicen a tus agentes cómo usar tecnologías específicas. Se cargan automáticamente.

## Cómo funcionan

```
Tú: "@dev Crea un componente React con Tailwind"
     ↓
Agente detecta .tsx → Carga react-19
Agente detecta Tailwind → Carga tailwind-4
     ↓
Resultado: código que sigue los patrones correctos
```

## Ver skills disponibles

```bash
ywai skills
```

## Crear un skill personalizado

1. Crear carpeta en tu proyecto
2. Escribir `SKILL.md` con instrucciones
3. Registrar con `ywai skill-registry`
