# Skills Reference

Referencia completa de todos los skills que ywai instala. Cada skill le da a tus agentes conocimiento especializado sobre una tecnología.

## Frontend

### react-19

**Trigger:** Componentes React, hooks, JSX

**Qué enseña:**
- Patrones de React 19 con React Compiler
- Server Components y Server Actions
- Hooks modernos (use, useOptimistic, useFormStatus)
- Manejo de estado y efectos

**Ejemplo:**
```
@dev Crea un componente React para el formulario de login
→ El agente sigue los patrones de react-19 automáticamente
```

### angular-core

**Trigger:** Componentes Angular, signals, inject, zoneless

**Qué enseña:**
- Componentes standalone
- Signals y computed
- Dependency injection con `inject()`
- Zoneless change detection

### angular-architecture

**Trigger:** Estructura de proyectos Angular

**Qué enseña:**
- Organización de módulos y features
- Lazy loading de rutas
- Patrones de arquitectura escalable

### angular-forms

**Trigger:** Forms, validación en Angular

**Qué enseña:**
- Reactive Forms vs Template-driven
- Validación custom
- Manejo de estado del formulario

### angular-performance

**Trigger:** Optimización de Angular, imágenes, lazy loading

**Qué enseña:**
- OnPush change detection
- TrackBy en ngFor
- Lazy loading de módulos
- Optimización de imágenes

### tailwind-4

**Trigger:** Estilos, CSS utilities, responsive design

**Qué enseña:**
- Utilidades de Tailwind CSS 4
- Responsive design con breakpoints
- Theming y dark mode
- Custom plugins

### yz-ui

**Trigger:** Yoizen UI components, design system

**Qué enseña:**
- Design system de Yoizen
- Paleta de colores y tipografía
- Componentes predefinidos
- Tokens de diseño
