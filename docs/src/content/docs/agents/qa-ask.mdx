# QA Ask

## Qué hace

Responde preguntas sobre testing — explica frameworks, patrones, y mejores prácticas.

## Cuándo usarlo

| Tarea | Ejemplo |
|---|---|
| Preguntas | `@qa-ask ¿Cómo mockeo API calls en Vitest?` |
| Frameworks | `@qa-ask ¿Cuál es la diferencia entre unit e integration?` |
| Mejores prácticas | `@qa-ask ¿Cómo testeo error handling?` |

## Límites

| ✅ Puede | ❌ No puede |
|---|---|
| Responder preguntas | Escribir tests completos (`@qa-dev`) |
| Explicar frameworks | Revisar código (`@qa-reviewer`) |
| Dar ejemplos | Explorar codebase (`@qa-finder`) |
