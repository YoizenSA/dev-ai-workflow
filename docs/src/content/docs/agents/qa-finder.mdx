# QA Finder

## Qué hace

Busca código relacionado con testing — encuentra archivos, patrones, y código sin cobertura.

## Cuándo usarlo

| Tarea | Ejemplo |
|---|---|
| Buscar tests | `@qa-finder Find all tests related to auth` |
| Código sin cobertura | `@qa-finder What code has no tests?` |
| Patrones | `@qa-finder How do we write tests here?` |

## Límites

| ✅ Puede | ❌ No puede |
|---|---|
| Buscar archivos | Escribir tests (`@qa-dev`) |
| Identificar gaps | Revisar tests (`@qa-reviewer`) |
| Explorar patrones | Diseñar estrategia (`@qa-analyst`) |
