# Migration Orchestrator

Coordina **migraciones de código complejas** entre frameworks. Divide la migración en páginas, delega a los agentes correctos, y ensambla los resultados.

<Badge text="Grupo: social-refactor" variant="note" />

## Cuándo usarlo

| Tarea | Ejemplo |
|---|---|
| Migrar framework | "Migrate this AngularJS app to React" |
| Refactoring grande | "Refactor the entire UI layer to use the new design system" |
| Migración incremental | "Migrate page by page, not all at once" |

## Lo que hace

1. **Clasifica** el alcance de la migración (usa `@migration-scope`)
2. **Planifica** el orden de migración (usa `@migration-planner`)
3. **Delega** la implementación página por página
4. **Valida** la paridad funcional (usa `@migration-validator`)
5. **Coordina** dependencias entre páginas

## Agentes que coordina

| Agente | Rol |
|---|---|
| `@migration-scope` | Clasifica qué hay que migrar |
| `@migration-planner` | Crea el plan de migración |
| `@migration-validator` | Valida paridad funcional |
| `@migration-validator-focused` | Valida una página específica |

<Aside type="tip">
Este agente es para migraciones grandes y complejas. Para cambios simples, usa `@dev` directamente.
</Aside>
