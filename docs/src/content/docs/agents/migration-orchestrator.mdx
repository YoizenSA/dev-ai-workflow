# Migration Orchestrator

## Qué hace

Coordina migraciones de código entre frameworks. Divide la migración en páginas y valida cada una.

## Cuándo usarlo

| Tarea | Ejemplo |
|---|---|
| Migrar framework | "Migra de Angular a React" |
| Refactoring grande | "Refactoriza toda la UI" |
| Migración incremental | "Migra página por página" |

## Flujo

1. `@migration-scope` clasifica qué hay que migrar
2. `@migration-planner` crea el plan
3. `@dev` implementa la migración
4. `@migration-validator` valida paridad

## Límites

| ✅ Puede | ❌ No puede |
|---|---|
| Coordinar migración | Implementar código (`@dev`) |
| Delegar a agentes | Crear planes (`@migration-planner`) |
| Validar resultados | Clasificar alcance (`@migration-scope`) |

<Aside type="tip">
Para migraciones pequeñas (1-2 archivos), usá `@dev` directamente.
</Aside>
