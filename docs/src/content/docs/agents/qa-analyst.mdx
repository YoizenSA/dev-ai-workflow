# QA Analyst

## Qué hace

Diseña estrategias de testing y analiza cobertura.

## Cuándo usarlo

| Tarea | Ejemplo |
|---|---|
| Estrategia | `@qa-analyst Diseña estrategia para pagos` |
| Cobertura | `@qa-analyst ¿Qué cobertura tenemos en auth?` |
| Priorización | `@qa-analyst ¿Qué tests escribir primero?` |

## Límites

| ✅ Puede | ❌ No puede |
|---|---|
| Diseñar estrategia | Escribir tests (`@qa-dev`) |
| Analizar cobertura | Explorar codebase (`@qa-finder`) |
| Priorizar | Revisar tests (`@qa-reviewer`) |
