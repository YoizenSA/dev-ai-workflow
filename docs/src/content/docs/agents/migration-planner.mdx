# Migration Planner

## Qué hace

Crea planes de migración detallados con dependencias y estrategia de rollback.

## Cuándo usarlo

| Tarea | Ejemplo |
|---|---|
| Planificar migración | "Plan the migration from Angular to React" |
| Analizar dependencias | "What pages depend on the old auth module?" |

## Límites

| ✅ Puede | ❌ No puede |
|---|---|
| Crear planes | Implementar código |
| Analizar dependencias | Clasificar alcance (`@migration-scope`) |
| Documentar riesgos | Validar paridad (`@migration-validator`) |
