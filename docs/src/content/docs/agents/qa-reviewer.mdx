# QA Reviewer

## Qué hace

Revisa la calidad de los tests escritos por `@qa-dev`.

## Cuándo usarlo

| Tarea | Ejemplo |
|---|---|
| Revisar tests | `@qa-reviewer Revisa tests del módulo de pagos` |
| Verificar cobertura | `@qa-reviewer ¿Los tests cubren todos los edge cases?` |
| Calidad | `@qa-reviewer ¿Las assertions son significativas?` |

## Límites

| ✅ Puede | ❌ No puede |
|---|---|
| Revisar tests | Escribir tests (`@qa-dev`) |
| Identificar tests frágiles | Explorar codebase (`@qa-finder`) |
| Sugerir mejoras | Diseñar estrategia (`@qa-analyst`) |
