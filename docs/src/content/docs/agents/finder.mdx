# Finder Agent

## Qué hace

Busca archivos, entiende la estructura del proyecto, y encuentra código relevante.

## Cuándo usarlo

| Tarea | Ejemplo |
|---|---|
| Buscar archivos | `@finder Find all test files related to auth` |
| Entender estructura | `@finder Show me the project structure` |
| Encontrar código | `@finder Where is the login logic?` |
| Explorar dependencias | `@finder What files import UserService?` |

## Límites

| ✅ Puede | ❌ No puede |
|---|---|
| Buscar archivos | Modificar archivos |
| Explicar estructura | Escribir código |
| Encontrar código | Ejecutar comandos |

## Cuándo usarlo vs otros

| Situación | Agente |
|---|---|
| "¿Dónde está X?" | `@finder` |
| "¿Cómo funciona X?" | `@ask` |
| "Implementá X" | `@dev` |

<Aside type="tip">
`@finder` es ideal como primer paso antes de delegar a `@dev` o `@architect`.
</Aside>
