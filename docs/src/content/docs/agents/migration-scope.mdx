# Migration Scope

## Qué hace

Clasifica qué hay que migrar — identifica archivos, módulos, y dependencias.

## Cuándo usarlo

| Tarea | Ejemplo |
|---|---|
| Clasificar alcance | "What needs to change to migrate to the new API?" |
| Auditar dependencias | "Which files import the old auth module?" |

## Límites

| ✅ Puede | ❌ No puede |
|---|---|
| Escanear codebase | Crear planes (`@migration-planner`) |
| Clasificar archivos | Implementar código |
| Identificar dependencias | Validar paridad (`@migration-validator`) |
