# Migration Validator Focused

Valida **una página específica** de la migración — versión enfocada del validator general.

<Badge text="Grupo: social-refactor" variant="note" />

## Cuándo usarlo

| Tarea | Ejemplo |
|---|---|
| Validar una página | "Validate only the /login page migration" |
| Revisión rápida | "Quick check on the migrated component" |

## Lo que hace

1. **Valida** solo la página objetivo (no toda la migración)
2. **Verifica** paridad funcional de esa página
3. **Reporta** findings específicos

## Lo que NO hace

- ❌ NO valida toda la migración (eso es `@migration-validator`)
- ❌ NO modifica código
- ❌ NO expande el alcance de validación
