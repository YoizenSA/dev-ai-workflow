# Memory Agent

## Qué hace

Gestiona la memoria persistente de tu proyecto. Guarda decisiones, descubrimientos, y contexto.

## Cuándo usarlo

| Tarea | Ejemplo |
|---|---|
| Buscar decisiones | `@memory ¿Qué decidimos sobre auth?` |
| Buscar bugs | `@memory ¿Qué bugs arreglamos en pagos?` |
| Buscar patrones | `@memory ¿Qué patrones usamos para error handling?` |

## Tipos de memoria

| Tipo | Qué guarda | Ejemplo |
|---|---|---|
| `decision` | Decisiones técnicas | "Elegimos PostgreSQL sobre MongoDB" |
| `bugfix` | Bugs y soluciones | "Fixed N+1 query en UserList" |
| `pattern` | Patrones | "Usamos repository pattern" |
| `config` | Configuración | "JWT expiry = 15min" |

## Cuándo se activa automáticamente

- Al **empezar** sesión: recupera contexto
- Al **terminar** sesión: guarda resumen
- Al **tomar** decisión: la registra
- Al **arreglar** bug: documenta causa y solución

<Aside type="tip">
`@memory` se activa automáticamente. No necesitás invocarlo manualmente para guardar decisiones.
</Aside>
