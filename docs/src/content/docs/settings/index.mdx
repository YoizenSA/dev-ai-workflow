# Settings UI

## Qué es

Un panel de configuración visual donde gestionás ywai sin tocar archivos.

## Quick path

```bash
# 1. Abrir
ywai kanban

# 2. Ir a Settings
http://localhost:5768/settings
```

## Pestañas

| Pestaña | Qué configurás |
|---|---|
| **General** | Directorio de trabajo, agente predeterminado |
| **Agentes** | Perfiles, permisos, skills vinculados |
| **Skills** | Ver disponibles, activar/desactivar |
| **Memoria** | Ver observaciones, buscar, limpiar |
| **MCP** | Servidores conectados, endpoints |
| **SDD** | Specification-Driven Development |
| **ADO** | Azure DevOps, PAT tokens |
| **Git** | Estado del repo, hooks |
| **Push** | Notificaciones |

## Settings UI vs CLI

| Situación | Settings UI | CLI |
|---|---|---|
| Configurar por primera vez | ✅ | |
| Cambio rápido | ✅ | |
