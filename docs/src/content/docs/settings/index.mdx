# Settings UI

El Settings UI es un **panel de configuración visual** donde puedes gestionar la configuración de ywai sin tocar archivos.

## Acceder a Settings

```bash
ywai kanban
```

Luego navega a `http://localhost:5768/settings`.

## Pestañas disponibles

### General

Configuración general de ywai:
- Directorio de trabajo por defecto
- Agente predeterminado
- Preferencias de logging

### Agentes

Gestiona los perfiles de agentes:
- Ver agentes instalados
- Editar system prompts
- Configurar permisos de herramientas
- Vincular skills

### Skills

Gestiona los skills instalados:
- Ver skills disponibles
- Activar/desactivar skills por agente
- Ver triggers de cada skill

### Memoria

Gestiona la memoria persistente:
- Ver observaciones guardadas
- Buscar en la memoria
- Limpiar entradas antiguas

### MCP

Configura servidores MCP:
- Ver servidores conectados
- Configurar endpoints
- Verificar estado de conexión

### SDD

Configuración de Specification-Driven Development:
- Habilitar/deshabilitar SDD
- Configurar rutas de specs
- Gestionar el registro de capabilities

### ADO

Configuración de Azure DevOps:
- Conectar cuenta de ADO
- Configurar proyectos
- Gestionar PAT tokens

### Git

Configuración de Git:
- Ver estado del repositorio
- Configurar hooks
- Gestionar credenciales

### Push

Configuración de notificaciones push:
- Habilitar notificaciones
- Configurar canales
- Gestionar suscripciones

<Aside type="tip">
El Settings UI es la forma más fácil de configurar ywai. Para configuración avanzada o automatización, usa los comandos CLI.
</Aside>

## Acceso remoto

El Settings UI está disponible en `http://localhost:5768`. Si necesitas acceso remoto:

```bash
# Escuchar en todas las interfaces
ywai kanban --host 0.0.0.0
```

<Aside type="caution">
No expongas el Settings UI en producción sin autenticación. Es una herramienta de desarrollo.
</Aside>
