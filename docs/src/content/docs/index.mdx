## ¿Qué es ywai?

ywai configura tu agente AI (opencode, Claude Code, Cursor, etc.) con **agentes especializados** y **skills** que saben cómo usar tu stack tecnológico. Un solo comando y tu agente se convierte en un equipo de desarrollo completo.

<CardGrid>
  <Card title="Agentes especializados" icon="star">
    Dev, QA, Architect, Reviewer, DevOps — cada uno con un rol claro y límites definidos.
  </Card>
  <Card title="Skills preinstalados" icon="puzzle">
    React, Angular, TypeScript, Tailwind, .NET, DevOps, y más. Se cargan automáticamente.
  </Card>
  <Card title="Un solo comando" icon="rocket">

## Antes vs Después

| Sin ywai | Con ywai |
|---|---|
| Agente genérico para todo | Especialistas por tarea |
| Configurar skills manualmente | `ywai install` y todo queda listo |
| Buscar skills para cada framework | Decenas de skills preinstaladas |
| Mantener configuraciones por separado | Un solo comando para actualizar todo |

## Empezar ahora

```bash
# Instalar
curl -fsSL https://github.com/Yoizen/dev-ai-workflow/releases/latest/download/install.sh | bash

# Configurar
ywai install

# Verificar
ywai doctor

# Usar en opencode
@dev Implementa el endpoint de login
```
