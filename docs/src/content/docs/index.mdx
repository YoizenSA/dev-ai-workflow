**ywai** es un CLI wrapper alrededor de [gentle-ai](https://github.com/Gentleman-Programming/gentle-ai) que añade:

- Skills adicionales no incluidas en gentle-ai (React 19, Angular, Tailwind 4, TypeScript, Biome, .NET, Playwright, DevOps, git-commit)
- Instalación y actualización con un solo comando
- Detección automática de 15+ agentes AI (OpenCode, Claude Code, Cursor, Gemini CLI, etc.)

## ¿Por qué ywai?

| Sin ywai | Con ywai |
|---|---|
| Configurar skills manualmente en cada agente | `ywai install` y todo queda listo |
| Buscar skills para cada framework | Decenas de skills preinstaladas |
| Mantener configuraciones por separado | Un solo comando para actualizar todo |
