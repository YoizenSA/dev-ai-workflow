## El patrón común

Todas las guías siguen el mismo patrón:

```
1. Vos le decís al orchestrator qué necesitás
2. El orchestrator elige los agentes correctos
3. Los agentes hacen su trabajo
4. El orchestrator te reporta el resultado
```

No necesitas saber qué agente hace cada cosa. El orchestrator se encarga de distribuir.
