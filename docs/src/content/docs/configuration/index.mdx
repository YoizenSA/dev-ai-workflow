# Configuración

## Estructura

```
~/.config/opencode/
├── agents/          # Perfiles de agentes
├── skills/          # Skills instalados
└── commands/        # Commands slash
```

## Perfiles de agentes

Cada agente tiene 3 archivos:

| Archivo | Qué contiene |
|---|---|
| `AGENT.md` | System prompt (comportamiento) |
| `tools.json` | Permisos de herramientas |
| `skills.txt` | Skills vinculados |

## Personalizar un agente

<Steps>

1. Encontrar el perfil en `~/.config/opencode/agents/`
2. Editar `AGENT.md` para cambiar comportamiento
3. Editar `tools.json` para cambiar permisos
4. Editar `skills.txt` para agregar/quitar skills
5. Reiniciar opencode

</Steps>

## Configuración del proyecto

### AGENTS.md

Creá un `AGENTS.md` en la raíz de tu proyecto para dar contexto:

```markdown
# AGENTS.md

## Convenciones
- TypeScript estricto
- Commits convencionales
- Tests con Vitest

## Estructura
- src/components/ — Componentes React
- src/services/ — Lógica de negocio
```

### Skills personalizados

```
mi-proyecto/.agents/skills/mi-skill/SKILL.md
```

Registrar: `ywai skill-registry --cwd /mi/proyecto`

## Verificar

```bash
ywai doctor
```

<Aside type="tip">
Si usás opencode, la configuración se instala automáticamente en `~/.config/opencode/`.
</Aside>
