# Primeros pasos

## Quick path

<Steps>

1. **Instalar ywai**
   ```bash
   curl -fsSL https://github.com/Yoizen/dev-ai-workflow/releases/latest/download/install.sh | bash
   ```

2. **Configurar tu agente**
   ```bash
   ywai install
   ```

3. **Verificar**
   ```bash
   ywai doctor
   ```

4. **Abrir opencode y usar**
   ```
   @dev Hola, ¿qué podés hacer?
   ```

</Steps>

<Aside type="tip">
ywai detecta automáticamente qué agente AI estás usando (opencode, Claude Code, Cursor, etc.) y configura todo en el lugar correcto.
</Aside>

## Tu primera tarea

| Qué pedir | Agente | Ejemplo |
|---|---|---|
| Pregunta simple | `@ask` | `@ask ¿Cómo funciona JWT?` |
| Código nuevo | `@dev` | `@dev Crea un endpoint GET /api/health` |
| Tests | `@qa` | `@qa Escribe tests para UserService` |
| Decisión de diseño | `@architect` | `@architect ¿Monolith o microservices?` |
| Feature completa | Orchestrator | `Implementa auth con JWT` |

| Concepto | Qué es | Ver más |

| **Agentes** | Especialistas que trabajan como equipo | [Agentes →](/dev-ai-workflow/agents) |

## Comandos esenciales

| Comando | Qué hace |
|---|---|
| `ywai install` | Instala todo (agentes + skills + config) |
| `ywai update` | Actualiza a la última versión |
| `ywai doctor` | Verifica que todo funcione |
| `ywai skills` | Lista los skills disponibles |
| `ywai kanban` | Abre el tablero visual |
