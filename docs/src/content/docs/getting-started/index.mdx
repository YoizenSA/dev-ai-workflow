# Primeros pasos

Instala ywai y empieza a usar agentes especializados en minutos.

## Instalación

<Steps>

1. **Instala ywai**

   ```bash
   curl -fsSL https://github.com/Yoizen/dev-ai-workflow/releases/latest/download/install.sh | bash
   ```

2. **Instala la configuración en tu agente**

   ```bash
   ywai install
   ```

   Esto configura:
   - Agentes especializados (dev, qa, architect, reviewer, devops, ask)
   - Skills para tu stack tecnológico
   - System prompts optimizados

3. **Verifica que todo esté bien**

   ```bash
   ywai doctor
   ```

4. **Abre opencode y empieza a usar agentes**

   ```
   @dev Hola, ¿qué puedes hacer?
   ```

</Steps>

<Aside type="tip">
La instalación detecta automáticamente qué agente AI estás usando (opencode, Claude Code, Cursor, etc.) y configura todo en el lugar correcto.
</Aside>

## Tu primera tarea

Una vez instalado, prueba con algo simple:

```
@ask Explica la estructura de este proyecto
```

Luego algo más complejo:

```
@dev Crea un endpoint GET /api/health que retorne { status: "ok" }
```

## Conceptos clave

### Agentes

Son especialistas que trabajan como un equipo. Cada uno tiene un rol claro:

| Agente | Para qué |
|---|---|
| `@dev` | Escribir código, arreglar bugs |
| `@qa` | Tests y validación |
| `@architect` | Decisiones de diseño |
| `@reviewer` | Code review |
| `@devops` | CI/CD e infraestructura |
| `@ask` | Preguntas e investigación |

[Ver documentación completa de agentes →](/agents)

### Skills

Son guías especializadas que le dicen a los agentes cómo usar tecnologías específicas. Se cargan automáticamente — no necesitas configurar nada.

[Ver documentación completa de skills →](/skills)

### Kanban

Tablero visual para monitorear el progreso cuando delegas tareas a múltiples agentes.

```bash
ywai kanban
```

[Ver documentación completa del Kanban →](/kanban)

## Comandos esenciales

| Comandos | Qué hace |
|---|---|
| `ywai install` | Instala todo (agentes + skills + config) |
| `ywai update` | Actualiza a la última versión |
| `ywai doctor` | Verifica que todo funcione |
| `ywai skills` | Lista los skills disponibles |
| `ywai kanban` | Abre el tablero visual |

[Ver todos los comandos →](/commands)
