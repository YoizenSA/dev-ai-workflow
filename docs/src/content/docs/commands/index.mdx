# Comandos CLI

## Quick reference

| Comando | Qué hace | Cuándo usarlo |
|---|---|---|
| `ywai install` | Instala agentes + skills + config | Primera vez |
| `ywai update` | Actualiza gentle-ai + skills | Después de actualizar |
| `ywai upgrade` | Actualiza el binario de ywai | Nueva versión de ywai |
| `ywai doctor` | Verifica la instalación | Si algo no funciona |
| `ywai skills` | Lista skills disponibles | Ver qué hay instalado |
| `ywai agents` | Lista agentes detectados | Verificar configuración |
| `ywai skill-registry` | Refresca registro de skills | Después de modificar skills |
| `ywai kanban` | Abre el tablero visual | Monitorear agentes |
| `ywai serve` | Inicia servidor como servicio | Servidor always-on |

## Detalles

### `ywai install`

Instala gentle-ai, skills y la configuración completa.

```bash
ywai install                    # Instalación completa
ywai install --dry-run          # Ver qué cambiaría
ywai install --agent opencode   # Solo para un agente
ywai install --preset minimal   # Preset específico
```

**Presets:**

| Preset | Qué instala |
|---|---|
| `full-gentleman` | Todo (default) |
| `ecosystem-only` | Solo gentle-ai, sin skills extra |
| `minimal` | Solo lo esencial |

### `ywai update`

Actualiza gentle-ai, re-sincroniza skills y regenera el registro.

```bash
ywai update
```

**Qué hace:**
1. Actualiza gentle-ai
2. Re-sincroniza skills
3. Regenera registro de skills
4. Verifica enlaces

### `ywai doctor`

Verifica que la instalación esté correcta.

```bash
ywai doctor
```

**Qué verifica:**
- gentle-ai instalado y actualizado
- Skills correctamente enlazados
- Rutas de configuración válidas
- Sin conflictos entre agentes

### `ywai serve`

Inicia el servidor de control como servicio de fondo.

```bash
ywai serve           # Iniciar
ywai serve --stop    # Detener
ywai serve --status  # Ver estado
```

## Flags comunes

| Flag | Descripción | Ejemplo |
|---|---|---|
| `--agent, -a` | Agente específico | `ywai install --agent opencode` |
| `--dry-run` | Vista previa | `ywai install --dry-run` |
| `--cwd` | Directorio | `ywai skill-registry --cwd /mi/proyecto` |
| `--preset` | Preset | `ywai install --preset minimal` |

<Aside type="tip">
Siempre corre `ywai doctor` después de instalar o actualizar.
</Aside>
