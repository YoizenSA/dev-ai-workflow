# React Code Review Checklist

## React
- [ ] Functional components only
- [ ] Hooks rules respected (top-level only)
- [ ] `key` prop on list items
- [ ] `alt` text on images

## TypeScript
- [ ] No `any` type
- [ ] Props properly typed

## Styling
- [ ] Tailwind classes preferred
- [ ] No inline styles
- [ ] Responsive design considered

## Standards
- [ ] Components ≤ 80 lines
- [ ] Files ≤ 400 lines
