# React Engineering Constitution

## Stack
- React 19, TypeScript (strict), Tailwind CSS 4
- Vite (recommended)

## Critical Rules
- Functional components only (no class components)
- Hooks rules: no hooks inside loops/conditions
- No `any` — use proper types or `unknown`
- Use Tailwind for styling (no inline styles, minimal CSS modules)
- Images must have `alt` text
- Components ≤ 80 lines, files ≤ 400 lines
